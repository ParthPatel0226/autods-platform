View AI reasoning

Imputation strategy: Median was chosen for Age because the distribution is right-skewed (skew = 0.39), and 19.9% missingness is too high for listwise deletion. KNN imputation would be marginally more accurate but adds compute cost for minimal gain.

Encoding decisions: Label encoding for Sex (binary, 2 values) avoids the dummy trap. Ordinal encoding for Pclass because (1, 2, 3) have natural order. One-Hot for Embarked (3 unordered categories, low cardinality).

Domain features: Charlson Index flagged because the healthcare domain config detected potential comorbidity columns. family_size and Age × Pclass interactions came from the tool registry's healthcare recommendations and known survival-model patterns.

Outliers: IQR capping for Age and Fare preserves extreme but plausible values (infant passengers, first-class fares) while reducing leverage in linear models.
