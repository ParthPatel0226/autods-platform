Infrastructure

Deployment Options

Local-first with zero infrastructure. Deploy anywhere with Docker.

● Local Development

Quickest path. Python venv + Makefile commands.

git clone <repo> cd autods-platform make setup cp .env.example .env # add API key make run # localhost:8501

● Docker Compose

Single container. DuckDB/SQLite/MLflow embedded.

docker-compose up -d # Dashboard: localhost:8501 # API: localhost:8000

● Cloud Platforms

Streamlit Cloud (free), HuggingFace Spaces, AWS/GCP/Azure containers.

# Streamlit Cloud: point to dashboard/app.py # HuggingFace: Streamlit Space type # Docker: any container host
